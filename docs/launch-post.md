## LinkedIn

I open-sourced SignalToPost, a small tool I built for my own writing workflow.

The idea is simple:

- capture raw thoughts in Telegram
- turn GitHub activity into content ideas
- generate X and LinkedIn drafts with a configurable voice
- review, rewrite, approve, or schedule everything from chat

It is intentionally self-hosted rather than a SaaS. Bring your own Telegram bot, OpenAI key, Postgres database, GitHub repos, and optional X credentials.

Stack: Next.js, TypeScript, Prisma, Postgres, OpenAI, Telegram, GitHub webhooks, Vercel, Neon.

I built it because the hard part of posting consistently is not only writing. It is reducing the friction between noticing something useful and turning it into a draft worth reviewing.
